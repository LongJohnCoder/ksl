void ksl_dc(const double thetai, double dc[2])¶

compute double precision direction cosines from an angle specified in radians

Parameters
  • thetai: [in] input angle
  • dc: [out] direction cosines, with sine stored in dc[0] and cosine is stored in dc[1]
